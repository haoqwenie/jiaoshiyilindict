## 三、需要注意的：

##### 1.乾卦读音修改为为gan卦，因为与谦卦冲突

##### 2.坤卦与困卦冲突，但无法修改，因此造成唯一的一个读音两个卦象的情况。

## 四、如何矫正的？

原始版到最终版之间使用了vlookup函数矫正，得到文件：4.最终版易林全文可导入输入法，可以直接导入到输入法中使用
